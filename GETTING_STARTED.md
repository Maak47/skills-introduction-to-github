# Getting Started with GitHub

Welcome! You're going through the GitHub Skills Introduction course. This guide provides quick reference information to help you as you learn.

## Current Course Progress

This repository is a hands-on course that teaches you GitHub basics. The course automatically tracks your progress and updates the README with new instructions as you complete each step.

## Course Overview

You'll learn:
1. **Creating a branch** - Make a parallel version of your code
2. **Committing changes** - Save your work with a descriptive message  
3. **Opening a pull request** - Propose your changes to be merged
4. **Merging a pull request** - Integrate your changes into the main branch

## Quick Reference

### Git Commands
```bash
# Check which branch you're on
git branch

# Switch to a branch
git checkout branch-name

# See your changes
git status
git diff

# Stage and commit changes
git add filename
git commit -m "Your message"

# Push changes to GitHub
git push origin branch-name
```

### GitHub Workflow
1. **Branch** - Create a separate line of development
2. **Commit** - Save changes with a message describing what you did
3. **Pull Request** - Ask to merge your changes
4. **Review** - Others can comment on your changes  
5. **Merge** - Incorporate approved changes into main
